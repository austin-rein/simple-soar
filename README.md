# A simple SOAR API
Personal project to develop an API based SOAR application

## ROADMAP:
- IP data enrichment:
    - VirusTotal
    - AbuseIPDB
    - GreyNoise
- Domain data enrichment:
    - VirusTotal
- Hash data enrichment:
    - VirusTotal
- Automated actions:
    - Firewall level block
    - Endpoint level block
- Report tracking DB (PostgreSQL)
- Report cache (Valkey)
- Alert platforms:
    - Slack
    - Mattermost
    - Teams
    - PagerDuty(?)
    - Signal
